# KYO ERP

Sistema ERP interno para gestión financiera y operativa de KYO.

## Objetivo

Centralizar la gestión financiera de la empresa desde el flujo de caja.

El sistema se basa en conciliación bancaria donde cada movimiento se
imputa a uno o varios rubros que determinan el resultado financiero.

## Módulos actuales

### Banco
Gestión de cuentas bancarias y conciliación.

### Movimientos
Registro de movimientos bancarios.

### Rubros
Clasificación contable de ingresos y egresos.

### Proveedores
Gestión de proveedores vinculados a pagos.

### Finanzas
Motor financiero del sistema.
